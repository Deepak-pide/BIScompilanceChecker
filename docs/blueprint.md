# **App Name**: BIS Smart Compliance Checker

## Core Features:

- Check Product Compliance: Allows users to input product details (name, description, image/PDF) and checks compliance against BIS standards using AI analysis to identify relevant rules.
- Compliance Report Generation: Generates a detailed compliance report indicating passed and failed requirements, along with a compliance score. Users can generate and save PDF reports. AI tool will reason about if information is incorporated.
- BIS Standard Search: Enables users to search for specific BIS standards by product category or keywords and view detailed rules.
- Saved Reports Management: Allows users to save and manage previously checked compliance reports for easy access and review.
- BIS Standards Information: Provides general information about BIS, its standards, and the importance of certification.

## Style Guidelines:

- Primary color: BIS Blue (#005BBB) for a professional and trustworthy feel.
- Background color: Light gray (#F4F4F4), providing a clean and modern backdrop.
- Accent color: BIS Orange (#FF7700) used for calls to action and highlighting important information.
- Font: 'Inter', sans-serif, for both headlines and body text, providing a clean and readable experience. 
- Use clear, professional icons to represent different functions and categories throughout the app.
- Card-based layout for displaying information in a structured and organized manner.
- Loader animation to indicate loading and processing states, enhancing user experience.
